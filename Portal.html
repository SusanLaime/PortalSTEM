Distinct Algorithm Cards

Explore different algorithms with concise descriptions and examples.

Binary Search
Efficiently finds the position of a target value within a sorted array by repeatedly dividing the search interval in half.
Example: Find 7 in [1, 3, 5, 7, 9]
Steps: Check middle (5), then right half (7), found!
Dijkstra's Algorithm
Finds the shortest path between nodes in a graph, which may represent, for example, road networks.
Example: Shortest path from A to D
Steps: A → B (2), B → D (4), total cost = 6
Merge Sort
A divide and conquer algorithm that splits the array into halves, sorts them, and merges the sorted halves.
Example: [4, 2, 5, 1]
Split: [4,2] [5,1] → Sort: [2,4] [1,5] → Merge: [1,2,4,5]
Depth-First Search (DFS)
Explores as far as possible along each branch before backtracking, used in tree and graph traversal.
Example: Traverse tree
Steps: Visit root → left subtree → right subtree
Quick Sort
Selects a 'pivot' and partitions the array into elements less than and greater than the pivot, recursively sorting the partitions.
Example: [3, 6, 2, 8]
Pivot: 3 → [2] [3] [6,8] → Sort recursively
